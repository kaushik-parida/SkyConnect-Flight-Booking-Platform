# SkyConnect – Flight Booking Platform

## Project Overview
SkyConnect is a scalable, microservices-based flight booking platform with core services for user authentication, booking, flight search, inventory, API gateway, and a responsive frontend.

## Tech Stack
- Backend: Java, Spring Boot, Spring Cloud
- Frontend: React + Tailwind CSS
- Database: MySQL
- Messaging: Kafka (optional)
- Deployment: AWS EC2/RDS
